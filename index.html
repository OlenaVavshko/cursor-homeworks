• git config --global user.name blabla - налаштування імені користувача
  • git config --global user.email blabal@bla - налаштування електронної адреси
  • git init - створення нового репозиторію
  • git status - перегляд статусу
  • git add - додавання файлів в репозиторій
  • git commit - збереження змін
  • git log - перегляд історії змін файлів проекту
  • git checkout - зміна гілок в репозиторії
  • git branch - створення, перелік, перейменування та видалення гілок
  • git push - внесення змін з віддаленого репозиторію в локальний
  • git pull - завантаження всіх змін локальної гілки у віддалений репозиторій
  • git merge - завантаження історії з віддаленого репозиторію і об'єднання з локальною
  • git clone - створення нового локального репозиторію із заданим ім'ям
Gitignore - це прорстий текстовий файл, в кожній стрічці якого міститься шаблок файла або каталога,який необхідно проігнорувати.
